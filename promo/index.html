FEATURES

核心功能

不只是翻译，更是一套完整的学术论文阅读体验。

📐

排版 100% 保真

公式、表格、图注、参考文献——翻译后排版和原文一模一样。不会出现错位、重叠、格式丢失。这是我们和其他工具最大的区别。

📖

三种阅读模式

Side by Side 左右对照 · Dual 双语交替 · Mono 纯译文。一键切换，随心阅读。内置缩略图导航、缩放、翻页。

🌍

20+ 翻译引擎

Google · DeepL · OpenAI · DeepSeek · 通义千问 · 腾讯 · Azure · Ollama 本地模型……按需选择，API Key 安全存储在本地。

🧠

AI 智能版面检测

基于 ONNX 深度学习模型，自动识别正文、公式、图表、页眉页脚，精准定位翻译区域，不翻译不该翻译的内容。

📚

术语库 & 提示词

自定义专业术语映射，确保 "Transformer" 始终翻译为 "Transformer"。支持 CSV/JSON 导入，搭配自定义提示词，翻译更精准。

⚡

多线程 & 分块翻译

最高 32 线程并发，50 页长文档自动分块翻译。再长的论文也不怕超时，进度条实时显示。

HOW TO USE

三步开始翻译

下载即用，无需安装 Python，无需配置环境。

1

选择文件

拖拽 PDF 到窗口，或点击选择文件。支持任意学术论文 PDF。

2

配置翻译

选择翻译引擎和目标语言。填写 API Key（免费引擎无需 Key）。选择输出格式。

3

一键翻译

点击"开始翻译"，等待进度条跑完。翻译完成后自动进入阅读模式，即刻对照阅读。
